== Description ==

Video Gallery Demo - http://www.apptha.com/demo/?product=videocomments

Video Gallery Home Page - http://www.apptha.com/category/extension/Wordpress/Video-Comments

Contus Video comments for wordpress is easy to install plugin which enables you to post video comments instead of text comments.It requires Red5 or FMS to store the video comments.
It is very simple, you just need to provide the server path from the wordpress settings page.

* Demo version available with WebFLVRecorder logo.
* Support Red5 and FMS server.
* Act as both recorder and as player. Play the recorded video with inbuilt player from WebFLVRecorder.
* It uses the standard comment structure of wordpress.
* Easy to setup and very user friendly.
* Streams without delay through Red5 while recording. When finished, it stores immediately without taking much time to upload.
* Flexible enough to modify upload location.
* Scalable to your own size.
* Capable of selecting the preferred video / audio device detected by the computer.
* Re-record option.

== Installation ==

= Minimal Requirements =
* PHP: 5.x.x
* mySQL: 4.0, 4.1 or 5.x
* WordPress: 2.0 or newer

= Recommended Requirements =
* PHP: 5.2.x or newer
* mySQL: 5.x
* WordPress: 2.8 or newer

= Basic Installation =
1. Download and unzip the current version of the Contus Video Comments Plugin.
2. Transfer the entire contus-video-comments directory to your /wp-content/plugins/ directory
3. Activate the Contus Video Comments plugin through the 'Plugins' menu in WordPress
4. That's it! You're done.


== Other Notes ==

1. After activating the plugin Go to Settings->Contus Video Comments 
2. You will find the settings to change the rtmp server location(You should specify one).
3. You will also have the other options like adjustable player,recorder width and height.
4. Maximum duration(in seconds) that a comment video can be.

== Frequently Asked Questions ==
